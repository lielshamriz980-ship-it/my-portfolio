## Design Pattern Standards Applied

### Common Sections Across All Pages:
1. **Hero Section**
   - RTL direction support
   - Large bold title (clamp 48-88px)
   - Descriptive subtitle
   - Dual CTAs (Primary blue + Outline)
   - Gradient background with blurred circles

2. **Stats Section**
   - 3 stat cards with icons
   - Large numbers with context
   - Semi-transparent white background with colored borders

3. **Feature Sections**
   - 2, 3, or 4-column grids
   - Icon + title + description format
   - Numbered badges (01, 02, etc.)
   - Alternating white and gradient backgrounds

4. **CTA Section**
   - Gradient background with primary color
   - White text
   - Large typography
   - Button with secondary action

### Color Palette Used:
- **Blue:** #2563EB / #1E40AF
- **Emerald:** #059669 / #10B981
- **Orange:** #F97316 / #FB923C
- **Red:** #DC2626 / #EF4444
- **Purple:** #7C3AED / #A78BFA
- **Indigo:** #4F46E5 / #6366F1
- **Cyan:** #0891B2 / #06B6D4
- **Amber:** #D97706 / #B45309
- **Dark Text:** #1E1B4B
- **Light Text:** #6B7280

### Technical Standards:
- RTL (Right-to-Left) Hebrew support with `direction: "rtl"`
- Responsive design with Tailwind classes
- Mobile-first approach with md: breakpoints
- Lucide React icons for all elements
- Inline styles for colors and gradients
- Proper spacing with clamp() for typography
- Semantic HTML structure
